## Déployer le site localement en 3 commandes

Prérequis :

- Node.js 16 ou plus
- Git
- VSCode ou votre éditeur de texte favori

`git clone https://github.com/ismbks/fablab`
`cd fablab`
`npm install`
`npm run docs:dev`
